## [0.1.0] - 2026-05-11

### 🎮 首次发布 - 植物大战僵尸复刻版

完整的冒险模式 1-1 到 1-10 关卡实现。

### 新增

#### 核心游戏系统
- **5×9 网格草坪**: 经典种植区域布局，支持鼠标点击种植
- **阳光经济系统**: 天空自动掉落阳光（每10秒），初始 50 阳光，每颗 25 单位
- **卡片冷却系统**: 每种植物独立冷却计时，冷却期间卡片半透明遮罩
- **铲子工具**: 点击激活铲子后可移除已种植的植物
- **割草机系统**: 每行一台割草机，僵尸到达左侧时自动触发清理全行
- **波次管理系统**: 每关多波僵尸，每波间隔递减，最后一波旗帜僵尸带头

#### 植物 (8种)
- **豌豆射手** (100☀): 向前方发射豌豆，检测行内僵尸后射击
- **向日葵** (50☀): 周期性产出阳光（每24秒）
- **坚果墙** (50☀): 高血量防御（4000HP），外观随血量变化显示裂缝
- **寒冰射手** (175☀): 发射冰豌豆，命中后减速僵尸50%，持续5秒
- **樱桃炸弹** (150☀): 种植0.5秒后爆炸，对3×3范围造成1800伤害
- **大嘴花** (150☀): 吞噬前方单个僵尸（1800伤害），咀嚼42秒后恢复
- **双发射手** (200☀): 每次攻击连续发射两颗豌豆（间隔150ms）
- **土豆地雷** (25☀): 埋入地下15秒激活，激活后踩踏即爆炸（1800伤害）

#### 僵尸 (5种)
- **普通僵尸**: 200HP，慢速移动
- **旗帜僵尸**: 200HP，略快速度，标志大波来袭
- **路障僵尸**: 560HP，路障帽提供额外护甲
- **撑杆僵尸**: 340HP，快速移动，可跳跃越过第一个植物
- **铁桶僵尸**: 1300HP，铁桶提供高额护甲

#### 场景 (5个)
- **BootScene**: 资源加载与进度条
- **MenuScene**: 主菜单，标题动画与开始按钮
- **LevelSelectScene**: 2×5 关卡选择网格，已解锁/未解锁状态
- **GameScene**: 核心游戏主循环，含种植、战斗、波次、胜负判定
- **VictoryScene**: 过关动画，星星特效与下一关按钮

#### 关卡数据 (10关)
- **1-1**: 教程关，仅普通僵尸，3波
- **1-2**: 引入向日葵和旗帜僵尸，4波
- **1-3**: 引入坚果墙和路障僵尸，4波
- **1-4**: 引入寒冰射手和撑杆僵尸，5波
- **1-5**: 引入樱桃炸弹和铁桶僵尸，5波
- **1-6**: 引入大嘴花，6波
- **1-7**: 引入双发射手，7波
- **1-8**: 引入土豆地雷，7波
- **1-9**: 全植物可用，8波
- **1-10**: 最终挑战，9波，最终波包含14个僵尸

#### SVG 美术资源
- 9个植物 SVG（含 CSS 动画：摇摆、闪烁、火焰等）
- 5个僵尸 SVG（含行走、进食动画）
- 投射物 SVG（豌豆、冰豌豆、阳光）
- UI 元素 SVG（割草机）
- 所有资源使用程序化图形绘制，零外部图片依赖

#### 视觉效果
- 植物种植入场动画（缩放弹出）
- 僵尸行走摇摆动画
- 豌豆击中爆炸特效
- 阳光脉冲与飞行动画
- 坚果墙血量阶段变化
- 樱桃炸弹屏幕震动
- 僵尸减速蓝色色调
- 死亡消散动画

### 技术细节
- **引擎**: Phaser 3.80.1
- **渲染**: Canvas 2D + 程序化 Graphics 绘制
- **架构**: ES6 Class 继承体系
- **物理**: Phaser Arcade Physics（投射物移动、边界检测）
- **网格**: 自定义 GridManager 类管理 5×9 种植网格
